### 3. Implementasyon

#### `difficulty_score.py` (yeni)
- `compute_difficulty_score(level, min_steps)` — BFS metriklerine dayalı zorluk skoru.
- `score_to_difficulty(score)` / `difficulty_to_target_score(diff)` — Skor ↔ etiket dönüşümü.
- `ScoreConfig.for_target_score(target_score)` — Hedef skora göre `LevelConfig` varyantı seçimi.
- `TARGET_SET_PROFILE` — 12 seviyelik ideal zorluk eğrisi.
- `sort_set_by_difficulty()`, `set_profile_deviation()` — Set profili araçları.

#### `pipeline.py` (güncellendi)
- `generate_level()` artık `target_score` parametresi alıyor.
- `ScoreConfig.for_target_score(target_score).to_level_config()` ile parametre seçimi.
- Üretim sonrası:
  1. `maxCmds = min_steps + margin` (skor farkına göre margin ayarı)
  2. `stars = [min_steps, min_steps + margin - 1]`
  3. `level.difficulty = score_to_difficulty(compute_difficulty_score(level, min_steps))`
- `generate_set()` artık:
  1. `TARGET_SET_PROFILE`'den hedef skorları kullanıyor.
  2. Her seviyede `best_delta` takip ediyor (en yakın skora sahip seviyeyi seçiyor).
  3. Set üretildikten sonra `sort_set_by_difficulty()` ile monoton sıralama yapıyor.
  4. `level.difficulty`'leri gerçek skora göre yeniden hesaplıyor.

#### Plugin Güncellemesi
- `analyze_level.py`: `compute_difficulty_score` kullanıyor, `difficulty_mismatch` flag'i ekledi.
- `validate_level.py`: Zorluk uyarısı eklendi (`expected_diff != actual_diff`).
