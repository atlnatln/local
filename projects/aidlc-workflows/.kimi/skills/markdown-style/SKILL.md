# markdown-style

All content in this repository is Markdown. Follow the
`.markdownlint-cli2.yaml` configuration.

## Disabled Rules

| Rule                        | Reason Disabled                                     |
| --------------------------- | --------------------------------------------------- |
| MD013 (line length)         | Long URLs, tables, and code examples are acceptable |
| MD033 (inline HTML)         | `<img>` tags are used for screenshots               |
| MD024 (duplicate headings)  | Section names repeat across platform guides         |
| MD036 (emphasis as heading) | Bold text used as sub-labels in lists               |

## Enforced Rules

| Rule                         | Requirement                              |
| ---------------------------- | ---------------------------------------- |
| MD060 (table alignment)      | Table pipes must be vertically aligned   |
| MD040 (fenced code language) | Always specify a language on code fences |

## Commands

```bash
# Lint all markdown files
npx markdownlint-cli2 "**/*.md"

# Fix markdown lint issues automatically
npx markdownlint-cli2 --fix "**/*.md"
```

## Ignored Paths

- `node_modules/**`
- `.claude/**`
- `scripts/aidlc-evaluator/test_cases/**`
- `CHANGELOG.md` (auto-generated by git-cliff)
